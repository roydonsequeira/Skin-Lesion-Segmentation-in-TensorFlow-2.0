# Documentation & Screenshots

Add your project visuals here so the main README can display them on GitHub.

## Suggested contents

| File / folder      | Description |
|--------------------|-------------|
| `screenshots/`     | App UI (e.g. Streamlit), training curves, ROC-AUC |
| `architecture/`    | U-Net and ResU-Net diagrams (`u-net-architecture.png`, `RESUNET_ARCH.png`) |

## Using in README

- Root sample images: place `image.jpg` and `mask.png` in the repo root `img/` folder.
- U-Net results: place result images (e.g. `ISIC_0000012.jpg`) in `UNET/results/`.
- Architecture diagrams: place in `UNET/img/` and `RESUNET/img/` and they will show in the main README.

After adding images, commit and push; they will appear in the README on GitHub.
